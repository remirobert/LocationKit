Location wrapper around CoreLocation in swift.

## Requirements

* iOS 8
* Swift 1.2

## Installation

#### [CocoaPods](http://cocoapods.org)

comming soon, if you ask for it.

#### [Carthage](https://github.com/Carthage/Carthage)

````bash
github "remirobert/LocationKit"
````
then
```bash
carthage bootstrap
```
